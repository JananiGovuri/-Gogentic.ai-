# -Gogentic.ai-
GenAI Conversational Persona Platform (Gogentic.ai) 
Configurable multi-agent conversational AI platform built on LangGraph — supports few-shot personas, RAG-backed knowledge, multi-step tool-using agents, and LLM evaluation via Judge models and LangSmith.
Gogentic.ai is a production-grade platform for building and deploying configurable AI personas powered by LangGraph and LangChain. Each persona is defined via a YAML config with few-shot examples, function-calling tools, and RAG-backed knowledge retrieval — enabling dynamic, context-aware multi-turn conversations without code changes. 
The agent graph follows a planner
→ executor 
→ memory architecture, supporting complex reasoning, tool use, and persistent conversation state. An integrated evaluation system uses Judge LLMs and LangSmith to continuously score responses on accuracy, helpfulness, and cost — making it observable and production-ready from day one.
